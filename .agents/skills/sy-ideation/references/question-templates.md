# Ideation Question Templates

Use one question per round. Keep questions concrete.

## Goal Clarification

- What concrete outcome should this change produce?
- Which user/problem scenario is primary?
- What is the minimum success signal?

## Constraint Discovery

- Which parts are strictly out of scope this iteration?
- Any non-functional constraints (performance, reliability, compliance)?
- Any dependency or compatibility boundaries?

## Trade-off Decision

- If we optimize for speed, what quality risk is acceptable?
- If we optimize for maintainability, what delivery trade-off is acceptable?
- Which option has the simplest rollback boundary?
